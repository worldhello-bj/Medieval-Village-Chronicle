## Visual Differences

### Color Scheme
- **Hidden Endings (✨)**: Purple with pulsing animation
- **Special Endings (⭐)**: Golden Yellow
- **Victory**: Green
- **Destruction**: Red
- **Survival**: Yellow

### Special Achievement Box
Only appears for Special and Hidden endings:
- Gradient background (purple-amber)
- Gold border
- Achievement title
- Condition summary

### Animations
- Hidden endings: Pulsing text effect
- Special endings: Static golden color
- Basic endings: Standard colors
